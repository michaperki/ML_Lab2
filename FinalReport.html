Overview

Type of Bias Investigated

This lab aims to investigate age-related bias across different racial groups using the facial recognition model VGG16. More specifically, this lab will assess whether VGG16 shows racial bias when predicting ages based on race. The bias could result in overestimating or underestimating the ages of individuals from certain races.

Pre-Trained Model

The pre-trained model being investigated is VGG16, a widely used convolutional neural network (CNN) pre-trained on the ImageNet dataset. VGG16 is known for its strong feature extraction when it comes to facial recognition tasks, however, the dataset it was trained on may not have had full representation across different racial and age groups, resulting in an existing bias.

Relevance of This Investigation

  • Ethical & Fair AI Development: Biased age predictions can lead to unfair treatment in real-world applications such as security systems, hiring processes, and social media algorithms. Racial groups may be predicted as older or younger than they actually are, leading to discrimination in these areas.
  • Model Deployment Implications: Many image classification models rely on a CNN architecture like VGG16. If bias exists in this CNN architecture, it might appear in other image classification models that use a similar architecture.
  • Training Data Bias: Bias could also stem from the training dataset used for these image classification models. For example, VGG16 is trained on ImageNet, which may contain imbalanced representations of certain races and age groups, leading to systemic bias.

Research Question

Research Questions

  • Does VGG16 systematically overestimate or underestimate the ages of individuals based on race?
  • Are there significant differences in Mean Absolute Error (MAE) among the different racial groups?

Hypothesis

  • Null Hypothesis: The VGG16 model does not exhibit significant age estimation bias across different racial groups.
  • Alternative Hypothesis: The VGG16 model exhibits systematic bias where it consistently overestimates or underestimates the ages of individuals from specific racial groups.

Methods

  • Dataset: This lab uses the UTKFace dataset, which labels images with age, gender, and race. The dataset includes five racial categories: White, Black, Asian, Indian, and Others (including Hispanic, Latino, and Middle Eastern).
  • Baseline Model: A pre-trained VGG16 model will be fine-tuned for age estimation using the UTKFace dataset. Predictions will be compared to true ages to calculate the MAE for each racial group.
  • Statistical Analysis: An ANOVA test will be used to determine differences in MAE across racial groups. A p-value of less than 0.05 suggests significant differences, indicating racial bias in the model.

Bias Mitigation Technique

Reweighted Loss Function

The chosen bias mitigation technique modifies the loss function to reweight underrepresented groups. Traditional age estimation models use MAE, treating all samples equally. However, if the model produces higher errors for certain racial groups, a weighted loss function can penalize errors more heavily for these groups. This approach forces the model to pay more attention to biased predictions.

Measuring Bias Mitigation Technique

An ANOVA test will determine whether MAE significantly differs between racial groups after mitigation. If the adjusted model returns a p-value greater than 0.05, it suggests a reduction in bias.

Results

Baseline Model (Before Bias Mitigation)

The baseline VGG16 model exhibited systematic disparities in age prediction errors across racial groups.

MAE per Group:
  • White (Race 0): 27.53 years
  • Black (Race 1): 36.93 years
  • Asian (Race 2): 51.69 years
  • Indian (Race 3): 52.47 years
  • Others (Race 4): 45.99 years

The ANOVA test revealed a statistically significant difference in MAE across racial groups (p < 0.05), confirming the presence of bias.

Reformed Model (After Bias Mitigation)

The de-biased model showed reduced disparities in age prediction errors.

MAE per Group:
  • White (Race 0): 10.67 years
  • Black (Race 1): 10.15 years
  • Asian (Race 2): 6.18 years
  • Indian (Race 3): 7.78 years
  • Others (Race 4): 7.27 years

However, the ANOVA test still showed statistically significant differences in MAE (p < 0.05), indicating remaining bias despite improvements.

Hypothesis Evaluation

  • Null Hypothesis: Rejected, as the initial ANOVA test showed significant differences in MAE across groups.
  • Alternative Hypothesis: Supported, as the baseline model exhibited systematic bias. Bias mitigation reduced disparities but did not completely eliminate them.

Research Question Evaluation

  • Does VGG16 systematically overestimate or underestimate the ages of individuals based on race?
    Yes. The baseline model showed racial disparities in both overestimating and underestimating certain races.
  • Are there significant differences in MAE among different racial groups?
    Yes. The baseline model revealed disparities through the ANOVA test.

Conclusion/Limitations

Conclusion

This study investigated age-related bias in the VGG16 model across different racial groups. The analysis confirmed disparities in age prediction errors, with the model overestimating or underestimating ages based on race.

A reweighted loss function was implemented to address this bias by assigning higher penalties to groups with greater prediction errors. The adjusted model showed improvements, reducing disparities in MAE across racial groups. However, the ANOVA test indicated that some bias persisted.

This study highlights the importance of bias detection and mitigation in machine learning, particularly in facial recognition and age estimation. Implementing techniques like reweighted loss functions can help create more equitable AI models.

Limitations

  • Dataset Representation Bias: The effectiveness of bias mitigation depends on dataset diversity. Underrepresented groups may still experience hidden biases.
  • 'Other' Race Classification: The dataset groups Hispanic, Latino, and Middle Eastern individuals under 'Other,' leading to loss of data granularity.
  • Generalizability: The findings are based on a specific dataset and model, and may not generalize to other models or real-world conditions.

def load_and_preprocess_image(path, age, race, augment=False, age_scaler=100.0):
    """
    Loads and preprocesses an image.

    Parameters:
        path (str): File path to the image.
        age (int): Age label.
        race (int): Race label.
        augment (bool): Whether to apply data augmentation.
        age_scaler (float): Factor to normalize the age value.

    Returns:
        image: Preprocessed image tensor.
        label: Tensor with normalized age and race.
    """
    image = tf.io.read_file(path)
    image = tf.image.decode_jpeg(image, channels=3)
    image = tf.image.resize(image, [224, 224])
    if augment:
        image = tf.image.random_flip_left_right(image)
        image = tf.image.random_brightness(image, max_delta=0.1)
        image = tf.image.random_contrast(image, lower=0.9, upper=1.1)
    image = image / 255.0
    age_norm = tf.cast(age, tf.float32) / age_scaler
    label = tf.stack([age_norm, tf.cast(race, tf.float32)])
    return image, label

def create_dataset(df, batch_size=32, shuffle=False, augment=False):
    """
    Creates a TensorFlow dataset from a DataFrame.

    Parameters:
        df (DataFrame): DataFrame with 'file_path', 'age', and 'race' columns.
        batch_size (int): Batch size.
        shuffle (bool): Whether to shuffle the dataset.
        augment (bool): Whether to apply data augmentation.

    Returns:
        ds: Prepared tf.data.Dataset.
    """
    ds = tf.data.Dataset.from_tensor_slices((df['file_path'].values,
                                              df['age'].values,
                                              df['race'].values))
    ds = ds.map(lambda p, a, r: load_and_preprocess_image(p, a, r, augment),
                num_parallel_calls=tf.data.AUTOTUNE)
    if shuffle:
        ds = ds.shuffle(1000)
    return ds.batch(batch_size).prefetch(tf.data.AUTOTUNE)

def mae_metric_age_only(y_true, y_pred):
    """
    Computes the Mean Absolute Error (MAE) for age prediction.

    Parameters:
        y_true: True labels.
        y_pred: Predicted labels.

    Returns:
        MAE value.
    """
    return tf.reduce_mean(tf.abs(y_true[:, 0] - tf.squeeze(y_pred)))

def evaluate_model(model, ds, age_scaler=100.0):
    """
    Evaluates the model on a dataset and returns a DataFrame with true ages, predicted ages, and races.

    Parameters:
        model: Trained model.
        ds: Dataset to evaluate.
        age_scaler (float): Factor used to scale age predictions back.

    Returns:
        DataFrame with evaluation results.
    """
    true_ages, predicted_ages, races = [], [], []
    for images, labels in ds:
        preds = model.predict(images)
        true_ages.extend((labels[:, 0].numpy() * age_scaler).tolist())
        predicted_ages.extend((preds.flatten() * age_scaler).tolist())
        races.extend(labels[:, 1].numpy().astype(int).tolist())
    return pd.DataFrame({
        'True Age': true_ages,
        'Predicted Age': predicted_ages,
        'Race': races
    })

def plot_mae_comparison(initial_mae, fine_tuned_mae, filename):
    """
    Plots a bar chart comparing MAE per race between the initial and fine-tuned models.

    Parameters:
        initial_mae (dict): MAE values for the initial model.
        fine_tuned_mae (dict): MAE values for the fine-tuned model.
        filename (str): Path to save the figure.
    """
    labels = [f"Race {r}" for r in sorted(initial_mae.keys())]
    init_vals = [initial_mae[r] for r in sorted(initial_mae.keys())]
    ft_vals = [fine_tuned_mae[r] for r in sorted(fine_tuned_mae.keys())]
    x = np.arange(len(labels))
    width = 0.35
    fig, ax = plt.subplots(figsize=(8, 6))
    ax.bar(x - width/2, init_vals, width, label='Initial')
    ax.bar(x + width/2, ft_vals, width, label='Fine-Tuned')
    ax.set_ylabel('MAE')
    ax.set_title('MAE by Race: Initial vs. Fine-Tuned')
    ax.set_xticks(x)
    ax.set_xticklabels(labels)
    ax.legend()
    save_figure(fig, filename)

def plot_error_boxplots(init_df, ft_df, filename):
    """
    Creates boxplots comparing absolute errors between the initial and fine-tuned models for each race.
    Also performs t-tests for each race.

    Parameters:
        init_df (DataFrame): Results from the initial model.
        ft_df (DataFrame): Results from the fine-tuned model.
        filename (str): Path to save the figure.

    Returns:
        Dictionary with t-test results per race.
    """
    init_df = init_df.copy()
    ft_df = ft_df.copy()
    init_df['Absolute Error'] = np.abs(init_df['True Age'] - init_df['Predicted Age'])
    ft_df['Absolute Error'] = np.abs(ft_df['True Age'] - ft_df['Predicted Age'])
    init_df['Model'] = 'Initial'
    ft_df['Model'] = 'Fine-Tuned'
    combined = pd.concat([init_df, ft_df])
    combined['Race'] = combined['Race'].astype(str)
    fig, ax = plt.subplots(figsize=(10, 6))
    sns.boxplot(x='Race', y='Absolute Error', hue='Model', data=combined, ax=ax)
    ax.set_title('Error Distribution by Race and Model')
    save_figure(fig, filename)
    ttest_results = {}
    for r in sorted(combined['Race'].unique()):
        err_init = init_df[init_df['Race'] == int(r)]['Absolute Error']
        err_ft = ft_df[ft_df['Race'] == int(r)]['Absolute Error']
        if len(err_init) and len(err_ft):
            stat, p_val = stats.ttest_ind(err_init, err_ft, equal_var=False)
            ttest_results[r] = (stat, p_val)
    return ttest_results

def perform_anova_test(df, model_name):
    """
    Performs a one-way ANOVA test on the absolute errors across races.

    Parameters:
        df (DataFrame): Evaluation results.
        model_name (str): Name of the model.

    Returns:
        Tuple with the F-statistic and p-value.
    """
    df_copy = df.copy()
    df_copy['Absolute Error'] = np.abs(df_copy['True Age'] - df_copy['Predicted Age'])
    groups = [df_copy[df_copy['Race'] == r]['Absolute Error'] for r in sorted(df_copy['Race'].unique())]
    stat, p_val = stats.f_oneway(*groups)
    print(f"ANOVA for {model_name}: F-statistic = {stat:.2f}, p-value = {p_val:.3f}")
    return stat, p_val
